The Problem

Every AI framework runs tasks. None of them learn.

  • ✕ You discover a pattern, use it once, forget it next week
  • ✕ Same mistakes across different projects
  • ✕ No memory between sessions
  • ✕ Research findings disappear into bookmarks
  • ✕ No awareness of your hardware constraints

The Solution

Axiom learns from every build session.

  • ✓ Daily research distilled into spaced repetition briefs
  • ✓ Patterns detected in one project, synced to others
  • ✓ Wiki knowledge base with version control
  • ✓ Successful builds auto-extracted as reusable skills
  • ✓ VRAM budgeting, process registry, port management

14 Core Modules. One System.

Each module works independently or as part of the full stack. All open source. All configurable.

📚

Knowledge Accelerator

Daily web research across 11+ domains. Findings distilled into actionable briefs automatically.

🔁

Spaced Repetition

Key findings resurface at optimal intervals (day 1, 3, 7, 14). You actually retain what you learn.

⚙

Auto-Skill Extraction

Successful build patterns automatically become reusable skills. Your AI evolves from experience.

🛠

Self-Modification

Repeated corrections auto-promote to permanent rules. All through a staging system you control.

🔍

Gap Detector

Cross-references new patterns from research against your actual codebase. Finds what you're missing.

🔗

Cross-Project Sync

Detects patterns in one project that are missing from another. Keeps all projects at the same level.

🚨

Failure Replay

Captures failures with structured post-mortems. Periodically re-tests to prevent recurrence.

📝

Build Journal

Searchable history of everything you've built. Pattern indexing across projects and sessions.

🎛

Hardware-Aware

VRAM budgeting, process registry, port management. Knows your GPU limits before you hit them.

💾

System Snapshots

One-command portable backup of your entire AI config. Restore or migrate in seconds.

📑

Skill Registry

Searchable index of all your skills with usage tracking. Know what you have before building new.

📖

Wiki Knowledge Base

Git-versioned, Obsidian-compatible. Staging system ensures AI never overwrites your knowledge.

Layered Architecture

Layer 5: Learning — Knowledge Accelerator, Spaced Repetition, Gap Detector
Layer 4: Intelligence — Auto-Skill Extraction, Self-Modification, Cross-Project Sync
Layer 3: Operations — Process Registry, VRAM Management, Port Management
Layer 2: Knowledge — Wiki Vault, Build Journal, Staging System
Layer 1: Execution — Claude Code, Ollama, Gemini, GPT, any LLM
Layer 0: Hardware — GPU VRAM Budgeting, RAM Headroom, Disk Monitoring
