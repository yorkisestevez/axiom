// INSTALLATION

Running in 5 minutes.

Free, open source, zero external dependencies. Works on macOS, Linux, and Windows.

git clone https://github.com/yorkisestevez/axiom.git ~/.axiom && cd ~/.axiom && node setup.js // PASTE THIS
/ STEP 01

Clone the Repo

Clone Axiom into a hidden .axiom folder in your home directory. Requires Node 18+ and Git.

git clone https://github.com/yorkisestevez/axiom.git ~/.axiom
/ STEP 02

Run Setup Wizard

Auto-detects your GPU, Ollama models, and asks about your projects. Creates your config.

cd ~/.axiom && node setup.js
/ STEP 03

Verify Install

Runs a full system health check. You should see "ALL SYSTEMS NOMINAL."

node core/system-audit.js
/ STEP 04

Add Domain Packs

Optional: drop-in configs for specific verticals. Each pack includes its own SETUP.md.

node core/skill-registry.js scan

None of them learn.

  • Discover a pattern Monday. Forget it by Wednesday.
  • Same mistakes across every project you touch.
  • Zero memory between sessions. Zero context retained.
  • Research findings disappear into dead bookmarks.
  • No awareness of your GPU, VRAM, or running processes.

AXIOM learns from every build session.
Forever.

  • Daily research distilled into spaced repetition briefs.
  • Patterns detected in one project, synced to others.

  • Successful builds auto-extracted as reusable skills.

  • Repeated corrections auto-promote to permanent rules.

Auto-Skill Extraction

Self-Modification

Repeated corrections auto-promote to permanent rules through a staging system you control.

/ 05

Gap Detector

Cross-references research findings against your actual codebase. Finds what you're missing.

Failure Replay

Build Journal

Searchable history of everything you've built. Pattern indexing across projects and sessions.

/ 09

Hardware-Aware

VRAM budgeting, process registry, port management. Knows your GPU limits before you hit them.

/ 10

One-command portable backup of your entire AI config. Restore or migrate in seconds.

/ 11

Skill Registry

Searchable index of all your skills with usage tracking. Know what you have.

Wiki Knowledge Base

Git-versioned, Obsidian-compatible. Staging system ensures AI never overwrites your knowledge.
