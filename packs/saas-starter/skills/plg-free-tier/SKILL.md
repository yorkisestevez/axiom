# PLG Free Tier Design

## What
Product-led growth patterns where the free tier converts to paid at a natural usage threshold. No sales demos, no onboarding calls. The product sells itself.

## When to Use
Any SaaS priced under $300/mo targeting individual users or small teams.

## Core Pattern

1. Free tier: full feature access, limited by usage (not features)
2. Usage gate: clear threshold that maps to real value delivered
3. Upgrade prompt: appears at the moment of highest perceived value
4. Friction-free upgrade: one click, no sales call, instant access

## Key Rules
- Gate on usage, not features (users need to experience the full product)
- Show usage meter prominently so users see value accumulating
- Upgrade prompt at 80% of limit, not 100%
- Launch with one flat paid tier, expand later
