## Run It Yourself

### Scan one repo

```bash
npx axiom-check
```

### Generate a new Hall of Gaps report

Requires the GitHub CLI (`gh`) to be authenticated.

```bash
git clone https://github.com/yorkisestevez/axiom
cd axiom
node scripts/hall-of-gaps.js --topic nextjs --limit 50
```

The script will:
1. Use `gh` to fetch the top 50 repos tagged with `nextjs`
2. Shallow-clone each one to `scripts/hall-of-gaps-tmp/`
3. Run `axiom-check` against each
4. Aggregate findings across all 50
5. Write the report back to `content/hall-of-gaps.md`
6. Write a Reddit post draft to `content/reddit-post-hall-of-gaps.md`

### Custom topic

```bash
node scripts/hall-of-gaps.js --topic supabase --limit 25
node scripts/hall-of-gaps.js --topic claude-code --limit 30
```
